#002 Problem of the Week 172

A logic problem about tarts and the Queen of Hearts

The Problem

The Queen of Hearts had some tarts, but they were stolen.
Precisely one of the following statements about the Knaves of Clubs (C), Diamonds (D) and Spades (S) is true. Which one?

1. None of the the Knaves stole any tarts

2. The Knave of Clubs stole some tarts

3. Only one of the Knaves stole any tarts

4. At least one of the Knaves of Diamonds and the Knave of Spades stole no tarts

5. More than one of the three Knaves stole some tarts

Statement 1

Assuming its true:

  • Statement 2 is false, as C didnt steal any tarts
  • Statement 3 is false, as no Knaves stole any tarts
  • Statement 4 is true, as both D and S stole no tarts, so atleast one of them stole no tarts
  • Statement 5 is false, as more that one Knaves stole no tarts

As only 1 and 4 are true, it cannot be the only true statement

Statement 2

Assuming its true, there are three possibilities:

  • Only C stole: 2, 3, and 4 are true
  • C and only one other stole: 2, 4, and 5 are true
  • All three stole: 2 and 5 are true

As in every case more than one statement is true, so 2 cannot be the only true

Statement 3

Assuming its true, there are three cases:

  • Only C stole: 2, 3 and 4 are true
  • Only D stole: 3 and 4 are true
  • Only S stole: 3 and 4 are true

As in every case more than one statement is true, it cannot be the only true statement

Statement 4

Assuming its true, there are four cases:

  • Only D didnt steal: 4, 2 and 5 are true
  • Only S didnt steal: 4, 2 and 5 are true
  • Both C, D and S didnt steal : 1 and 4 are true
  • Only C stole: 2, 3 and 4 are true

As in every case more than one statement is true, it cannot be the only true statement

Statement 5

Assuming its true:

  • If C + S stole: 5, 2 and 4 are true
  • If D + S stole: only 5 is true
  • If C + D stole: 2, 4 and 5 are true
  • If everyone stole: 2 and 5 are true

THe statement is true as it works when D and S steal but C doesnt, as it has only one statement being true on its own

Reflection

I personally struggle with these questions as I always miss conditions (here I missed C + D stealing and it nearly messed up the whole thing) so i took my time and noted everythign possible. I liked how you had to notice that 5 was only true in a particular theft not every one making it more difficult to notice
